A tiny reading view for side projects is more about restraint than features.

Constraints

I capped the layout at 760px and kept typography at a relaxed 1.6–1.8 line height. Sidebars stay scrollable so the reading pane never jumps.

Fetching Content

The reader pulls local HTML files and injects them into the page. No markdown pipeline, no bundler. Just fetch, parse, and render.

// fetch and parse the article
const res = await fetch("posts/example.html");
const html = await res.text();
const doc = new DOMParser().parseFromString(html, "text/html");
article.innerHTML = doc.querySelector("article").innerHTML;

Why keep it simple?

Because it removes friction. I can drop any HTML snippet into /posts and it is live instantly.
